## Installation
 clone the repository:

   ```shell
   git clone https://github.com/arpnghosh/colabcoder.git
   ```
### Client
   ```shell
   cd client
   npm install
   npm run dev
   ```
### Server
   ```shell
   cd server
   npm install
   npm start
   ``````
